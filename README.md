# chromasmith-sandbox

Minimal React + Ladle sandbox for testing ForgeView live reload.

## Branches

- `main` - Initial repository with basic README
- `dev` - Full React + Ladle setup with components and stories

## Qick Start

Checkout the `dev` branch and run:

```bash
npm install
npm run ladle
```

The Ladle server will start on port 61001.

## Components

- `Button` - Simple button component
- `Card` - Card component with title and description
- `Hero` - Hero section component
